Setup Google Sheets Sync

Dashboard membaca harga kompetitor dari Google Sheets secara langsung. SKU yang ada di Sheets akan otomatis menggantikan data simulasi. SKU yang tidak ada di Sheets tetap menggunakan simulasi sebagai fallback.

Cara Publish ke Web

  1. Buka Google Sheets → klik menu File → Share → Publish to web.
  2. Di dropdown pertama pilih sheet harga_kompetitor.
  3. Di dropdown kedua pilih Comma-separated values (.csv).
  4. Klik tombol Publish, konfirmasi, lalu copy URL yang muncul.
  5. Paste URL di kolom input di dashboard, klik Sync Sekarang.

Logika Harga Otomatis

Dashboard menghitung Best Price dengan urutan prioritas berikut:

  1. Target Price = Rata-rata harga pasar (Shopee+Tokped+Blibli+Alfamart) × 1.05
  2. Jika margin < 5% → Best Price dinaikkan ke margin minimum 5% dari COGS
  3. Jika margin > 70% → Best Price diturunkan ke margin maksimum 70% dari COGS
Titik hijau (●) di tabel = harga real dari Google Sheets · Titik kuning (●) = harga simulasi. URL Sheets disimpan otomatis di browser — tidak perlu paste ulang setiap buka dashboard. Update harga di Sheets kapan saja, lalu klik Sync Sekarang untuk refresh.
